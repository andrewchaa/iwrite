# Rider keyboard shortcuts

| Command | Shortcut |
| :--- | :--- |
| Go to the previous place | ⌘\[ |
| Go to the next place | ⌘\] |
| Run all unit tests | ⇧⌘↵ |
| Select word and extend the selection | ⌥↑ |
| Shrink Selection | ⌥↓ |
| Show Build | ⌘0 |
| Show Git | ⌘9 |
| Show NuGet | ⌘7 |
| Show Terminal | ⌃⌘1 |
| Show Unit Test | ⌘8 |
