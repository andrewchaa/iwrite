# VS Code Shortcuts

| Windows | Mac | Description |
| :--- | :--- | :--- |
| alt + shft + mouse |  | column select |
| ctrl + \` |  | show terminal window |
| f12 |  | go to the definition |
| shft + alt +  **↓** |  | duplicate the line |
|  | ⇧ + ⌘ + space | show parameter hint |
| f2 |  | rename a symbol |
